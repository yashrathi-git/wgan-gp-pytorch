# WGAN-GP
Pytorch implementation of [Improved Training of Wasserstein GANs
](https://arxiv.org/abs/1704.00028)

## Examples
### MNIST (8 epochs)
* lr = 1e-4, betas = (0.0, 0.9), batch size = 32
